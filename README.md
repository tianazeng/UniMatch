# 🦁 UniMatch - Columbia University Edition


### 📖 About The Project
**UniMatch** is an intelligent, modern platform designed to help university students discover, match with, and manage their campus club applications. Inspired by the elegant aesthetic of Columbia University, it features a clean Columbia Blue (#C4D8E2) UI, glassmorphism effects, and an interactive AI matching experience. 

🌍 **Live Demo:** [(https://unimatch-ecru.vercel.app/)]

### ✨ Key Features
* 🔍 **Discovery Hub:** A comprehensive directory to explore, search, and filter campus clubs by categories (e.g., Tech, Arts, Sports).
* 🔮 **The Oracle (AI Matchmaker):** An interactive chat interface that asks tailored questions to find the perfect club for your weekend vibe.
* 📊 **Student Dashboard:** A personalized profile section to track application statuses (Pending, Interview, Accepted) with dynamic colored badges.
* 🔔 **Notification System:** Real-time mock alerts for club messages and interview reminders.
* 🔥 **Trending Leaderboard:** Stay updated with the most popular "Hot Picks" on campus.

### 🛠 Tech Stack
* **Framework:** Next.js 15 (React)
* **Styling:** Tailwind CSS
* **Icons:** Lucide-React
* **Deployment:** Vercel
